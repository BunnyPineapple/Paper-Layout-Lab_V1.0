# Paper Layout Lab ✿

A soft little printable page studio for planner lovers, notebook collectors, and people who want prettier paper without opening heavy design tools.

Paper Layout Lab helps you quickly generate printable notebook pages in different sizes, adjust line / grid / dot spacing, place an image, and export or print with ease.

---

## What is this?

Paper Layout Lab is a lightweight page generator made for people who love:

- planner inserts
- ring notebooks
- printable pages
- soft stationery-style interfaces
- customizable paper structures

Instead of building everything from scratch in a design tool, you can choose a notebook size, pick a page type, adjust spacing and margins, and generate a printable layout in just a few steps.

---

## Current features

### Notebook sizes
Currently supported:

- A6 Bible
- A6 FC
- A7
- M5
- A9
- Free size / custom size

### Page layout
Current main layout includes:

- one image area
- one writing area
- adjustable image size
- adjustable margins

This makes it easy to create a quick printable page with both visual space and writing space.

### Paper types
You can choose:

- Lined
- Grid
- Dot

### Spacing presets

#### Lined
- 6 mm
- 6.5 mm
- 7 mm
- 8 mm

#### Grid / Dot
- 2 mm
- 5 mm

### Output
Paper Layout Lab currently supports:

- direct printing
- PNG export

The print mode is designed to work better for real paper printing, especially when you want to print structure lines onto colored paper.

---

## Why I made this

I wanted a tool that feels lighter, softer, and more personal than a generic editor.

Most tools are either:
- too heavy
- too generic
- too manual
- or not really made for planner inserts

Paper Layout Lab is my attempt to make something that feels more like arranging a desk than building a document.
